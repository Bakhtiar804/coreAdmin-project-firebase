# 🚀 Firebase User Management System

A lightweight, secure, and real-time User Management System built with **HTML5, CSS3, JavaScript (ES6 Modules)**, and **Firebase (v10+)**. This project features user authentication, a real-time dashboard for managing user data, and dynamic UI states handled gracefully with **SweetAlert2**.

---

## ✨ Features

- **User Authentication:** - Email/Password Sign-Up and Login via Firebase Auth.
  - One-click **Google Sign-In** integration.
  - Automatic session routing (Protected routes using `onAuthStateChanged`).
- **Real-time CRUD Database:** - Add and store user profiles dynamically in **Cloud Firestore**.
  - Sync local tracking seamlessly with `localStorage`.
  - Live data updates inside the dashboard using Firestore's `onSnapshot`.
- **Enhanced User Experience (UX):**
  - **SweetAlert2 Loaders** for all async actions (fetching data, authenticating, saving, and deleting).
  - Secure delete operations with custom confirmation pop-ups.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Icons:** Font Awesome (v6.4.0)
- **Backend-as-a-Service (BaaS):** Firebase Auth & Cloud Firestore
- **Popups & Alerts:** SweetAlert2
